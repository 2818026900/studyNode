#  集合

##  List

主要包含三个实现：Arraylist 、 Linkedlist 、 Vector

###  Arraylist

1. 存放元素：有序
2. 线程安全：不安全
3. 数据结构：基于数组实现
4. 扩容大小：1,5倍

## Map

### Hashmap

- jdk7原理
  1. 计算键值hash
  2. 将hash取模数组长度
  3. 如果存在hash值重复，将重复值按照头插法链接到该位置
  4. 将头结点移动至下标位置
- jdk7与jdk8区别
  1. jdk8中链表长度大于8转红黑树，红黑树长度为6转链表
  2. 新节点插入链表时采用尾插法
  3. hash算法优化
  4. resize的逻辑修改（jdk7产生死锁场景）
- jdk8选用红黑树原因
  - 插入效率： 链表 >  红黑树  > 平衡二叉树
  - 查询效率： 平衡二叉树  > 红黑树  >  链表
  - hashmap既需要查询也需要考虑插入效率所以选用红黑树
